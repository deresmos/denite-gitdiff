denite-gitdiff
==
Denite source and kind for git diff


Requirements
--
* [vim-fugitive](https://github.com/tpope/vim-fugitive)
* Install gitpython
```bash
pip install gitpython
```


Installation
--
* To install using dein:
  ```
  [[plugins]]
  repo = 'deresmos/denite-gitdiff'
  ```


Usage
--
* `diffbranch` Comparing target branch.
